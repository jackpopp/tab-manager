# Tab Manager for chrome

A react based chrome extension for managing your tabs.

<img src="screenshot.png?raw=true" alt="Screenshot of the Tab Mananger" style="width:600px; margin: 0 auto;" />

## Build

```
npm install
npm run build
```

## Watcher

```
npm run watcher
```

## Dist version

```
npm run dist
```
